Projeto
Este repositório contém a resolução de exercícios organizados em duas partes. Abaixo, você encontrará instruções sobre como configurar e executar o projeto.
Para execução do projeto instale as bibliotecas listadas no arquivo requirements.txt

- Parte 1
A Parte 1 contém os seguintes exercícios:

    Exercício 1
    Descrição: Criada uma aplicação simples em FastAPI que utiliza o modelo GPT-2 da HuggingFace para gerar textos a partir de uma entrada fornecida via requisição HTTP.
    Execução: Entrar na pasta 'Parte 1' e no terminal executar o seguinte comando: uvicorn ex_1:app --reload
    
    Exercício 2
    Descrição: Criado um aplicativo FastAPI que utiliza o modelo de tradução Helsinki-NLP/opus-mt-en-fr da HuggingFace para traduzir textos do inglês para o francês.
    Execução: Entrar na pasta 'Parte 1' e no terminal executar o seguinte comando: uvicorn ex_2:app --reload

- Parte 2
A Parte 2 contém os seguintes exercícios:

    Exercício 1
    Descrição: Desenvolvovido um protótipo utilizando LangChain que simule um chatbot simples com Fake LLM.
    Execução: Entrar na pasta 'Parte 2' e no terminal executar o seguinte comando: uvicorn ex_1:app --reload
    
    Exercício 2
    Descrição: Desenvolvovido um aplicativo que utiliza LangChain para integrar a API da OpenAI.
    Execução: Entrar na pasta 'Parte 2' e no terminal executar o seguinte comando: uvicorn ex_2:app --reload
    
    Exercício 3
    Descrição: Criada uma API semelhante à Questão 2 (Parte 2), mas que utiliza o modelo Helsinki-NLP/opus-mt-en-de da HuggingFace para traduzir textos do inglês para o alemão.
    Execução: Entrar na pasta 'Parte 2' e no terminal executar o seguinte comando: uvicorn ex_3:app --reload
